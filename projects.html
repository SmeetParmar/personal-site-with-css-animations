Then after some time I came across one platform called Bootstrap. It is a very very useful tool for web designers because it is exactly like CSS, but its extended and simplified version wherein we can use CSS just by calling the specified class. For example, if you want to make a font larger and bolder, you will need to make seperate CSS and define everything there. But using bootstrap you just need to call "fs-1" class and it will do all things. That's the magic of bootstrap. In most of my project I used bootstrap.
